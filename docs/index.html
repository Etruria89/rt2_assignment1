Description :

This node advertise a position service. When the service is requested, a request containing minimum and maximum values for the x and y position is used to generate a random position between x (or y) min and x (or y) max.

State Machine¶

This files defines a finite state machine for the control of the holonomic robot.

Services :

/user_interface

Author

Federico Danzi

Version

0.1

Date

06/06/2021

Clients : /position_server

Action Client : /go_to_point

Description :

This node implements the finite state machine for the input on the control of the holonomic robot. It includes:

  • An Action client “go_to_point” that interacts with the action “Control2_1” to:

    • set a new goal

    • check if the goal has been reached

    • check if the goal has been cancelled

    • Define the new state

  • A Service server that advertise the command line user interface

  • A Service client that requires a new target postion from the /position_server
